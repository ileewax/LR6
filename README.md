### 14. Лог всех команд:
#### 14.1 git config - просмотр и настройка параметров Git
#### 14.2 git clone - клонирование репозитория 
#### 14.3 git pull - используется для извлечения и загрузки содержимого из удаленного репозитория
#### 14.4 git log --all - используется для просмотра истории всех коммитов
#### 14.5 git show - используется для просмотра подробных данных об объектах Git
#### 14.6 git merge - объединяет несколько последовательностей коммитов в общую историю 
#### 14.7 git push - передаёт в удалённый репозиторий изменения, сделанные в локальном репозитории
#### 14.8 git status - показывает состояния файлов в рабочем каталоге
#### 14.9 git add - перенос ожидающих изменений из рабочего каталога в раздел проиндексированных файлов Git
#### 14.10 git commit -m "Text" - делает для проекта снимок текущего состояния изменений, добавленных в раздел проиндексированных файлов
#### 14.11 git reset --hard - служит для отмены локальных изменений в разделе проиндексированных файлов и рабочем каталоге
#### 14.12 git branch "Text" - создает новую ветку с названием "Text"
#### 14.13 git checkout "Text" - переход на существующую ветку с названием "Text"
